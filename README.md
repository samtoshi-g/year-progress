## Tech
- Pure HTML/CSS/JS
- No dependencies (uses Inter font from Google Fonts)
- LocalStorage for birth year
- Real-time updates every second

## Features
- Year/month/week/day progress bars
- Days remaining for each
- Life progress calculator
- Birth year saved to localStorage
- Clean minimal design
- Existential quote footer
